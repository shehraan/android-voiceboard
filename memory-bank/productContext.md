# Product Context

## Problem Statement
Voice input on Android is fragmented. Users need a system-wide solution that works in any text field, not just specific apps.

## User Goals
1. Dictate quickly into any app (messaging, email, notes)
2. See real-time preview of transcription while speaking
3. Get clean, formatted output without manual editing
4. Trust that speech data stays on-device

## UX Principles
- Press-and-hold: most natural for precise control
- Mic-first UI: keyboard secondary to voice
- Immediate visual feedback: partial results visible
- Consistent formatting: Voice mode for raw, Message mode for polished

## Success Metrics ( qualitative for demo )
- Can complete a full dictation loop in under 5 seconds
- Text appears directly in focused field
- No cloud dependencies for core functionality
